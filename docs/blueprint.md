# **App Name**: Verdant Vision

## Core Features:

- Image Capture: Capture images of plants directly through the app.
- Plant Species Identification: Identify the species of the plant in the image using AI analysis.
- Disease Detection: Detect plant diseases from images. Provide descriptions of probable symptoms.
- Treatment Recommendations: Suggest treatments for detected plant diseases. This can also include dosage recommendations which will involve the use of a tool.
- Symptom Log: Save symptoms for the detected plant diseases to track the plants progress. Store these in a database
- User profile: Allow users to create profiles where they can log in and out to access their previous history.

## Style Guidelines:

- Primary color: Vibrant green (#77DD77) to evoke a sense of nature and health.
- Background color: Light, earthy brown (#E9E3D4), providing a natural, calming backdrop. 
- Accent color: Forest green (#228B22), analogous to the primary, adds depth and highlights key interactive elements. 
- Headline font: 'Alegreya', a humanist serif with a contemporary feel, suited for headlines or body text
- Body font: 'PT Sans', a humanist sans-serif combining modernity and warmth.
- Use plant-themed icons to represent different features and categories.
- Use a card-based layout to present information clearly and concisely.
